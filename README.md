# 🏪 Baqala Network

> A Web3 Telegram Mini App modernizing UAE Baqalas. Features AI-powered ordering, digital Hisaab (credit tabs), and crypto micro-settlements.

## 🌟 Features
- **🤖 AI Order Extraction**: Send a text to the Telegram bot to automatically parse items, prices, and profiles.
- **📒 Digital Hisaab (Credit Tabs)**: Manage your grocery tab natively inside Telegram. Sub-profiles supported (e.g., "Kids Snacks").
- **💳 Web3 Crypto Settlements**: Pay off your Hisaab using crypto to earn automatic discounts.
- **📍 Geolocation**: Discover and apply for credit at nearby Baqalas.
- **🏪 Vendor Dashboard**: Baqala owners can manage inventory, set custom crypto discounts, and approve customer credit limits.

## 🛠 Tech Stack
- **Frontend**: React.js, Vite, Ethers.js, Telegram Web App SDK
- **Backend**: Node.js, Express, Telegraf (Telegram Bot API), OpenAI API
- **Smart Contracts**: Solidity, Hardhat

## 🚀 Running Locally

**1. Clone the repository:**
git clone https://github.com/allan0/baqala.git
cd baqala-ecosystem

**2. Setup Backend:**
cd backend
npm install
npm start

**3. Setup Frontend:**
cd ../frontend
npm install
npm run dev
